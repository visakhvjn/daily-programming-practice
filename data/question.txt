<div class="question">
    <div class="title">String Compression Challenge</div>
    <div class="description">
        Implement a function that compresses a given string by replacing consecutive repeated characters with the character followed by the count of repetitions. For instance, the string "aaabbc" should be compressed to "a3b2c1". If the compressed string is not smaller than the original, return the original string.
    </div>
    <div class="examples">
        <strong>Examples:</strong>
        <pre>
        Input: "aaabbc"
        Output: "a3b2c1"

        Input: "abcd"
        Output: "abcd"

        Input: "aaaabbbbcdeee"
        Output: "a4b4c1d1e3"
        </pre>
    </div>
    <div class="constraints">
        <strong>Constraints:</strong>
        <ul>
            <li>The input string will have a length between 1 and 1000 characters.</li>
            <li>The input string will only contain lowercase letters (a-z).</li>
        </ul>
    </div>
    <div class="applications">
        <strong>Applications:</strong>
        <ul>
            <li>Data compression algorithms.</li>
            <li>Reducing storage size for text data.</li>
            <li>Improving the efficiency of text transmission over network protocols.</li>
        </ul>
    </div>
</div>
